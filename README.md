# The Midnight Grid

> A quiet suburb lit up its meters at 2 a.m. every night. I followed the load and found a story nobody on the grid expected.

The question: Why did a quiet neighbourhood spike its power use every night at 2 a.m.?
The result: A hidden overnight load worth a meaningful share of local peak.

## Run
    pip install -r requirements.txt
    python main.py

## Stack
Python, Prophet, Power BI (this demo isolates the overnight pattern directly).
